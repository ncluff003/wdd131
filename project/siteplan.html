Regex2HTML

The primary tool that I am endeavoring to create is the ability to convert a regular expression into a valid string for an HTML input pattern attribute. The other tools, and pages are also very nice, yet still are sub-objectives to the goal of providing something that is not able to be found anywhere else.

The site's purpose

This site will provide, first and foremost, a central tool that is not found anywhere else on the web. Also, it will give more access to other tools that are found, but still not as widely as they should be.
